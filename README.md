# Maher API (MVP)
Express.js API بذاكرة مؤقتة (in-memory) للتجربة السريعة.

## التشغيل
```bash
cd server
npm install
npm run dev
```
الخادم يعمل على: `http://localhost:4000`

## نقاط رئيسية
- تسجيل OTP وهمي: الكود دائمًا 1234
- تصنيفات جاهزة
- إنشاء طلب، تصفح الطلبات، إرسال عروض، قبول عرض
- لا يوجد دفع حقيقي في هذه النسخة (Stub)
